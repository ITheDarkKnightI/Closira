# 1. Встановлення додатку
Closira — Екранний перекладач Десктопний застосунок на базі Electron, призначений для захоплення виділеної області екрана з її подальшим перекладом.Швидкий стартДля початку роботи із застосунком виконайте такі команди в терміналі:Bash# 1. Встановлення залежностей
npm install

# 2. Запуск застосунку
npm start

# 3. Інструкція для користувача
Щоб здійснити переклад тексту з екрана, виконайте такі кроки:Натисніть кнопку «Виділити область екрана» в інтерфейсі застосунку або скористайтеся комбінацією клавіш Ctrl+Shift+T.
Після появи напівпрозорого оверлею виділіть за допомогою миші необхідну ділянку монітора.Застосунок автоматично виконає такі операції:
Обріже скріншот відповідно до заданих меж;
Розпізнає текст локально за допомогою бібліотеки Tesseract.js;
Здійснить автоматичний переклад отриманого тексту.
